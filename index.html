Introduction

What is ArnoldC?

ArnoldC is a unique and playful esoteric programming language inspired by famous quotations from Arnold Schwarzenegger's movies. Created by Lauri Hartikka in 2013, ArnoldC provides a fun and unconventional way to write code, using Arnold's iconic lines as commands.

Language History?

ArnoldC made its debut in 2013, offering a fresh take on programming languages by using Arnold Schwarzenegger's memorable phrases. The language quickly gained popularity among enthusiasts for its creative and humorous approach to coding.

Computational Class

ArnoldC's computational class is not clearly defined, as it falls into the category of esoteric languages primarily created for entertainment and experimentation. While it may not be suitable for practical programming tasks, it offers a unique way to explore programming concepts.

Getting Started

Installation

ArnoldC doesn't require installation, as you can write and run ArnoldC code using online interpreters. This eliminates the need for setting up a development environment, making it accessible to all.

Syntax & Commands

Keywords and Their Meanings

ArnoldC commands are based on Arnold Schwarzenegger's famous quotes and phrases. Here are some key commands and their meanings:

  • I LIED - False
  • NO PROBLEMO - True
  • BECAUSE I'M GOING TO SAY PLEASE - If
  • BULLSHIT - Else
  • YOU HAVE NO RESPECT FOR LOGIC - EndIf
  • STICK AROUND - While
  • CHILL - EndWhile
  • GET UP - Plus Operator
  • GET DOWN - MinusOperator
  • YOU'RE FIRED - MultiplicationOperator
  • HE HAD TO SPLIT - DivisionOperator
  • I LET HIM GO - ModuloOperator
  • YOU ARE NOT YOU YOU ARE ME - EqualTo
  • LET OFF SOME STEAM BENNET - GreaterThan
  • CONSIDER THAT A DIVORCE - Or
  • KNOCK KNOCK - And
  • LISTEN TO ME VERY CAREFULLY - DeclareMethod
  • GIVE THESE PEOPLE AIR - NonVoidMethod
  • I NEED YOUR CLOTHES YOUR BOOTS AND YOUR MOTORCYCLE - MethodArguments
  • I'LL BE BACK - Return
  • HASTA LA VISTA, BABY - EndMethodDeclaration
  • DO IT NOW - CallMethod
  • GET YOUR ASS TO MARS - AssignVariableFromMethodCall
  • HEY CHRISTMAS TREE - DeclareInt
  • YOU SET US UP - SetInitialValue
  • IT'S SHOWTIME - BeginMain
  • YOU HAVE BEEN TERMINATED - EndMain
  • TALK TO THE HAND - Print
  • I WANT TO ASK YOU A BUNCH OF QUESTIONS AND I WANT TO HAVE THEM ANSWERED IMMEDIATELY - ReadInteger
  • GET TO THE CHOPPER - AssignVariable
  • HERE IS MY INVITATION - SetValue
  • ENOUGH TALK - EndAssignVariable
  • WHAT THE FUCK DID I DO WRONG - ParseError

Variables and Capitalization

In ArnoldC, variables and keywords are differentiated by capitalization. Variables should always be written in lowercase, while keywords are in uppercase.

Code Structure

Maintaining proper code structure in ArnoldC is essential. Each keyword command should typically be written on a new line for clarity and readability.

Coding Conventions

When writing ArnoldC code, it's essential to follow coding conventions for clarity and consistency:

  • Variables should be in lowercase, while keywords should be in uppercase.
  • Each keyword command should be written on a new line to enhance readability.
